# STM32 Temperature Control System

## Overview
A thermostat and fan control system developed using STM32F401RE and LM35 temperature sensor.

## Features
- Real-time temperature measurement
- Automatic fan control according to temperature
- Temperature indication with LEDs
- Buzzer warning system
- Developed using STM32 HAL libraries

## Hardware Used
- STM32F401RE Nucleo Board
- LM35DZ Temperature Sensor
- LEDs
- Buzzer

## Development Environment
- STM32CubeIDE
- STM32 HAL Library
- C Programming Language
